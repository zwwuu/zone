# Zone for Hacker News

Zone is yet another Hacker News client, built with [Flutter], [Slidy], and [HNPWA API].

![home page][home]

## Setup
Clone this repo to your machine

**Requirements**
- Android SDK tools
- Android platform tools
- Dart
- Flutter
- Slidy

**Build**
```shell script
# install slidy
pub global activate slidy

# build apks for android
slidy run apk
```
